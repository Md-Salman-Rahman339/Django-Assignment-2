Variables declared inside a { } block cannot be accessed from outside the block: { let x = 2; } x can NOT be used outside the block

Tell us the use cases of null and undefined:


In simple terms, null is when we have an empty box on purpose, and undefined is when we have a box, but we haven't decided what to put in it or forgot to put something in it. Both are ways of saying "there's nothing meaningful here," but they're used in slightly different situations.
